Z3 is an SMT solver that we can use to understand the satisfiability of programs

**Satisfiability:** Where we determine if a formula **F** has a model
- If F is propositional, a model is a truth assignment to variables
$a \lor b \land c \implies a \land c \lor b \land c, (a = true, b = false, c = true)$
- if F is a first-order formula, model assigns values to variables and interpretation to all function and predicate symbols

### Using Z3

Z3 is based on the  [SMT-LIB](https://smt-lib.org/) syntax

**Important Z3 methods:**
- **Int(name)** creates an int const with a given name
- **FreshInt(name)** creates a unique const starting with name
- **IntVal(v), BoolVal(v)** creates integer and boolean values, usually used to include numbers/booleans in non-assignment declarations
- **+, -, /, <, <=, >, >=, etc** 
- **Distinct(a,b, ...)** ensures the arguments are distinct
- **And, Or, Not** 
- **For the z3 Solver class:**
	- **add(fml)** add formula to the solver
	- **check()** returns z3.sat, z3.unsat, or z3.unknown
	- **model()** model gets returned if the result is satisfiable
- **For the z3 Model class**
	- **eval(fml)** returns the value of fml in the model

**Example:**
```python
import z3

Day, (Mon, Tues, Wed, Thurs, Fri, Sat, Sun) = z3.EnumSort(
    "Day",
    ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"],
)
Animal, (Lion, Unicorn) = z3.EnumSort("Animal", ["Lion", "Unicorn"])

lie = z3.Function("lie", Animal, Day, z3.BoolSort())
yesterday = z3.Function("yesterday", Day, Day)

solver = z3.Solver()

# lying constraints
solver.add(
    lie(Lion, Mon),
    lie(Lion, Tues),
    lie(Lion, Wed),
    z3.Not(lie(Lion, Thurs)),
    z3.Not(lie(Lion, Fri)),
    z3.Not(lie(Lion, Sat)),
    z3.Not(lie(Lion, Sun)),
    lie(Unicorn, Thurs),
    lie(Unicorn, Fri),
    lie(Unicorn, Sat),
    z3.Not(lie(Unicorn, Sun)),
    z3.Not(lie(Unicorn, Mon)),
    z3.Not(lie(Unicorn, Tues)),
    z3.Not(lie(Unicorn, Wed)),
)

# yesterday function
solver.add(
    yesterday(Mon) == Sun,
    yesterday(Tues) == Mon,
    yesterday(Wed) == Tues,
    yesterday(Thurs) == Wed,
    yesterday(Fri) == Thurs,
    yesterday(Sat) == Fri,
    yesterday(Sun) == Sat,
)

today = z3.Const("today", Day)

# constraints from what the animals said today
solver.add(
    z3.Implies(lie(Lion, today), z3.Not(lie(Lion, yesterday(today)))),
    z3.Implies(z3.Not(lie(Lion, today)), lie(Lion, yesterday(today))),
    z3.Implies(lie(Unicorn, today), z3.Not(lie(Unicorn, yesterday(today)))),
    z3.Implies(z3.Not(lie(Unicorn, today)), lie(Unicorn, yesterday(today))),
)

result = solver.check()

if result == z3.sat:
    print(solver.model().eval(today))
else:
    print("Failed")
```
# Logical Equivalences

## De Morgan's Laws
- $¬(P \land Q) \equiv ¬P \lor ¬Q$
- $¬(P \lor Q) \equiv ¬P \land ¬Q$

## Implication Equivalences
- $P \to Q \equiv ¬P \lor Q$
- (Contraposition) $P \to Q \equiv ¬Q \to ¬P$

## Double Negation
- $¬(¬P) \equiv P$

## Distributive Laws
- $P \land (Q \lor R) \equiv (P \land Q) \lor (P \land R)$
- $P \lor (Q \land R) \equiv (P \lor Q) \land (P \lor R)$
